General

MasCN's introductions.

MasCN is an embedded systems, it can running in x86_64. And it's also a 64-bits operating system kernel.

It is development began in the Apr 22, 2022. It is an open-source project of GNU.

Why is this project name called MasCN ?

Becuease when I started to develop MasCN, I watched a movie about the dayly life.

So I named it MasCN.
